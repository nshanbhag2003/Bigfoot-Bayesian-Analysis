# Counting Sasquatch: A Bayesian Analysis of Reported Bigfoot Sightings

This is the project for **STAT GR5224: Bayesian Statistics** with **Professor Dobrin Marchev** at Columbia University.

This project uses Bayesian count regression to study whether reported Bigfoot sightings show meaningful temporal structure rather than random noise. Using a BFRO-based dataset from Kaggle, we compare Poisson and Negative Binomial models to analyze seasonality, overdispersion, and long-run trends in monthly sighting counts.

Dataset source: [Unlocking Mysteries of Bigfoot Through Sightings](https://www.kaggle.com/datasets/thedevastator/unlocking-mysteries-of-bigfoot-through-sightings)

Important Note: this repository does **not** prove Bigfoot exists, but it does suggest that Bigfoot reports are more statistically organized than Bigfoot himself.
